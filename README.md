## HB workshop

Repozitár k workshopu 16. septembra 2022.

Inštalácia aplikácie + Cypressu:
```npm install```

Štart aplikácie:
```npm start```

K správnemu fungovaniu workshopového repozitára je potrebné mať spustené dva procesy:
1. aplikáciu (`npm start`)
2. cypress (`npx cypress open`)
